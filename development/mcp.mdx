---
title: "MCP"
description: "Develop with the Pickaxe MCP server."
---

The Pickaxe CLI comes bundled with a local MCP server which runs on a `stdio` connection for development. It can be configured via the following config:

```json
{
  "mcpServers": {
    "pickaxe": {
      "command": "pickaxe",
      "args": ["mcp"]
    }
  }
}
```
